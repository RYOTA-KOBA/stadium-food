# README
【アプリ名】
スタジアムグルメ

【アプリ概要】
各ユーザーが自分が食べたスタジアムグルメを写真とともにコメントを添えて共有するサービスです。

【使い方】
ユーザー登録後、新規投稿ページから写真とコメント付きでスタジアムグルメを投稿する。
